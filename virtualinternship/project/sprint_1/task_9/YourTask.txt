В этом задании вы познакомитесь с файлами:
- build.gradle
- settings.gradle

Оба файла лежат в корне проекта.

В нашем репозитории мы используем многомодульный проект Gradle.

В нашем репозитории есть несколько отдельных модулей.
Все они прописаны в файле settings.gradle.
В этот файл вы добавляли и свой модуль.

Gradle модули при запуске общего процесса сборки (команда build)
собираются (билдятся) отдельно друг от друга (если между ними нет зависимостей).

В файле build.gradle расположенном в корне проекта есть общая конфигурация
для всех модулей проекта. Откройте этот файл и ознакомьтесь с его содержимым.

buildscript {
	repositories {
		mavenCentral()
	}
}
Эта часть настройки означает, что наш проект будет брать зависимости из репозитория Maven Central.
https://mvnrepository.com/repos/central
Это открытый репозиторий в котором публикуются Java библиотеки. Их можно подключать и
использовать в своем проекте.

	apply plugin: 'java'
	apply plugin: 'idea'
Эти строки означают, что мы используем Gradle плагины java и idea.
Узнайте больше про Gradle плагины.


	sourceCompatibility = '21'
	targetCompatibility = '21'
Означает, что мы пишем и компилим проект с использованием JDK 21.


	compileJava {
		options.encoding = "UTF-8"
	}

	compileTestJava {
		options.encoding = "UTF-8"
	}
Используем кодировку UTF-8.


	test {
		useJUnitPlatform()
	}
И платформу для тестирования JUnit 5.

Если вам не понятны какие-то вещи, загуглите инфу про них.
С некоторыми из них мы познакомимся в процессе разработки.

На этом наше знакомство с Gradle заканчивается и мы переходим к разработке.
